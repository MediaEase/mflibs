# info - v1.1.39
a library for information functions


### mflibs::info::ipv4::dig()

outputs IPv4 address via dig

#### return codes:

- 0 success
- 1 unable to resolve ip

#### example:

```bash
mflibs::info::ipv4::dig
```

### mflibs::info::ipv4::local()

outputs local IPv4 address

#### return codes:

- 0 success
- 1 unable to resolve ip

#### example:

```bash
mflibs::info::ipv4::local
```
